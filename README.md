# Lexia: Compliance. Refined.

Lexia is a precision digital audit toolkit designed to provide instant transparency into website compliance. It performs deep scans for privacy-relevant signals, focusing on GDPR requirements and data exposure.

## Key Features
- **Font Audit**: Detects external Google Fonts loading from third-party servers.
- **Tracker Detection**: Identifies active analytics scripts (Google Analytics, Meta Pixel, etc.).
- **CDN Analysis**: Scans for library loading via external CDNs.
- **Media Embeds**: Detects YouTube, Vimeo, and Google Maps integrations.
- **Legal Compliance**: Verifies the presence of Imprint/Legal Notice and Privacy Policy links.
- **Storage & Cookies**: Monitors local storage and cookie usage for consent mapping.

## Why Lexia?
Lexia is designed for web developers, privacy officers, and site owners who value clarity and professional aesthetics. It delivers a clean "Compliance Index" score to help optimize privacy standards instantly.

## Installation
Currently available for manual installation during development:
1. Clone this repository.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable "Developer mode".
4. Click "Load unpacked" and select the project folder.
